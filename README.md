# 🌐 Web Development Studies

Bem-vindo ao meu repositório de estudos focado no desenvolvimento web front-end! Este espaço foi criado para organizar todos os meus exercícios práticos à medida que domino **HTML5**, **CSS3** e **JavaScript**.

O objetivo é construir uma base sólida e limpa, documentando a minha evolução desde o início.

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização e layouts.
* **JavaScript:** Lógica e dinamismo no front-end.
